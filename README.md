# オムツ単価チェッカー

店舗でオムツの商品と値札を撮影し、枚数・価格・クーポン・CBを入れて1枚単価を比較するWebアプリです。PWA対応なので、スマホのホーム画面に追加してアプリ風に使えます。

## 起動

```powershell
node server.js
```

ブラウザで次を開きます。

```text
http://127.0.0.1:4173
```

同じWi-Fi内のスマホから確認したい場合は、PC側で次のように起動します。

```powershell
$env:HOST="0.0.0.0"; node server.js
```

表示された `LAN:` のURLをスマホで開きます。

ただし、スマホのカメラは原則としてHTTPS配信でないと使えません。外出先で本格的に使う場合は、HTTPS対応のホスティングに置いてください。

## 外出先で使う流れ

1. HTTPSのURLでスマホから開く
2. ブラウザメニューからホーム画面に追加
3. ホーム画面のアイコンから起動
4. 一度開いた後は、画面本体はオフラインでも起動できます

履歴はスマホ本体のブラウザ保存領域に残ります。サーバーには送信されません。

## GitHub Pagesで公開

1. GitHubで新しいリポジトリを作成
2. このフォルダ内のファイルをアップロード
3. リポジトリの `Settings` → `Pages` を開く
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main`、フォルダを `/root` にして保存
6. 表示された `https://ユーザー名.github.io/リポジトリ名/` をスマホで開く

公開後、スマホのブラウザメニューからホーム画面に追加すると、アプリ風に起動できます。

## できること

- ホーム画面追加
- オフライン起動
- カメラ撮影
- 商品名、サイズ、枚数、店頭価格、クーポン値引き、CB・ポイント還元の入力
- 実質価格と1枚単価の自動計算
- 同じ商品名・同じサイズの過去最安値との比較
- サイズ別の最安値一覧
- 履歴保存
- CSV書き出し

## 画像読み取りについて

ブラウザが `TextDetector` に対応している場合は、画像内テキストから金額・枚数・サイズの候補を反映します。未対応ブラウザでは、撮影画像を見ながら手入力してください。

履歴はブラウザの `localStorage` に保存されます。別端末・別ブラウザとは共有されません。
